How to add more games & icons

  1. Duplicate any <article class="game-card"> block inside .game-grid and update the slot number, title, and tags.
  2. Replace the emoji in <div class="game-icon">ðŸŽ®</div> with another emoji, a letter, or an SVG icon.
  3. If you want image icons, swap the icon div for an <img> tag, e.g. <img class="game-icon" src="icons/space.png" alt="Space Raiders" /> and put your icon images in an icons/ folder.
  4. To turn a slot into a real game link, wrap the card in <a class="game-card" href="/games/space-raiders.html">...</a> and add that game file.
  5. Add more soda themes by creating a new class in the animated-bg section (example: .bg-berry { background-image: linear-gradient(...); }) and applying it to a new .soda-card.
